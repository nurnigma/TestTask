# Пример выполнение тестового задания
-----------------
## Постановка задачи
Тествое задание звучало так:
Необходимо сделать docx-отчет по данным из xlsx-файла.
xlsx-файл (test.xlsx) содержит первичные данные.
Их необходимо сконвертировать либо в базу данных SQL, либо в xml-файл с определенной структурой.
На основании данных SQL или xml-файла нужно сформировать docx-отчет.
docx-отчет должен предоставлять информацию, о том сколько в предоставленных данных:
1. мужчин и женщин.
2. мужчин в возрасте 30-40 лет.
3. стандартных и премиум-аккаунтов.
4. женщин с премиум-аккаунтом в возрасте до 30 лет.

Рекомендуемый вариант реализации: консольное приложение с конфигурационным файлом (например, в нем указать путь до папки xlsx-файла данных, и пусть до папки docx-файла результатов) или веб-приложение с простейшим интерфейсом запроса пути до xlsx-файла с данными и выводом docx-файла результатов.
При демонстрации решения на проверку данные в файле будут меняться. Отчет будет должен формироваться на новых проверочных данных.
_______________________________________________________________
## Решение задачи
Мною были созданы методы чтение и записи под необходиме поля для конвертирования из xlsx в xml. 
Выбрать файл можно из диалогового окна :open_file_folder: и сохраняется xml файл рядом. Затем из диалогового окна можно написать название файла и место сохранения для docx.
